I am working as an Machine Learning Engineer at Engineerbabu working on an intersection of computer vision, biomedical and web development. Before that I completed my bachelors in computer science at Medicaps University. My research interest lies broadly in computer vision, especially generative models and adversarial learning. Much of my work includes image classification coupled with computer aided diagnosis and image generation. I've also explored seq2seq models for generating timeseries sequential data. Problems in this vein that I'm currently excited about include:

(1) Designing CNN architectures for Medical Image Segmentation
(2) Developing deep GANs for faster convergence and higher kernel density
(3) Mitigating adversarial attacks and their defenses

Projects

  • Cifar Wasserstien GAN

    A partially trained deep convolution generative adversarial networks(DCGAN) trained using Wasserstien strategy.

    All the implementation was done using keras and tensorflow, it took around 2500 iterations to show some comparabale results.

    Note: Due to limitations in compute resources model was only trained to above mentioned epochs. To get better results training can be resumed from current checkpoint.

  • Wasserstien GANs

    Powered by deep convolution generative adversarial networks(DCGAN) and backed by Wasserstien based learning strategy to generate fashion mnist images.

    All the implementation was done using keras and tensorflow, it provided comparabale results in around 100 EPOCHS

  • Denoising Auto Encoder

    Powered by Auto Encoders(AE) to denoise fashion mnist image dataset.

    All the implementation was done using keras and tensorflow, it provided a stellar results in just 10 EPOCHS

    Top two rows display noisy images whereas the last ones redenders reconstructed images corresponding to those noisy images

  • Fashion Image Generation using DCGAN

    Powered by deep convolution generative adversarial networks(DCGAN) to generate fashion mnist images.

    All the implementation was done using keras and tensorflow, it provided a stellar results in just 50 EPOCHS

  • Computer aided diagnosis for Pneumonia CT scan and Malaria blood cell slides

    Powered by convolution neural network(CNN) to classify lung CT scans into viral or bacterial pneumonia or normal and detecting malaria pathogens from blood sides.

    All the implementation was done using keras achieving an accuracy of 92% and 94% for pneumonia and malaria respectively.

    A flask based webapp was developed for serving the models

  • Covid Screening using lung CT scan images

    Powered by convolution neural network(CNN) and backed by transfer learning, MobileNetV2 architecture pretrained on imagenet was used to screen CovSars-2 virus.

    Two seperate models were trained one to tune only after 100 layers and other to tune complete architecture with AUC of 0.61 and 0.53 respectively.

  • Fashion Image generation using VAE

    A shallow variational autoencoder for generating fashion images based on fashion mnist which was successful in learning latent information from images

  • Language Translator: English to Hindi

    A shallow copy of GNMT for developing a real time machine translator for translating English language to Hindi. We achieved a BLEU score of 14.2 after successful training of a model.

  • Image Classification for Paleography Analysis

    Inspired by the story of In Codice Ratio an attempt was made to create an image classifier for predicting the medevial texts from image using a shallow CNN.

  • Generating Long articles like Wikipedia

    Developed a LSTM backed model for generating long sequence of text data based on wikidata of text. This project laid a foundation of my research inspiration for comparing various sequence-to-sequence models for generating tv scripts.

  • Face Detection

    A project aimed to detect human face from an image using support vector machines and HOG features. Overall accuracy of 78% was achieved after training a complete model.

  • Chatbot for Single QnA

    A chat bot was developed using bAbI dataset for answering a single contextual question.

  • Neural Style Transfer for generating artistic images

    We used VGG-16 architecture pretrained on imagenet dataset as a base model, and after tuning the model with the custom styles using gram matrix for filtering and calculating style of an image.

  • Text Classification(Sentiment)

    A two layered bidirectional GRU backed model intended to classify polarity of text on IMDB movie reviews.
